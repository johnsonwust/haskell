ジェネリックプログラミングはHaskellのデータ型を同型で統一的なデータ型に変換することで、複数のデータ型を統一的に扱えるような仕組みを提供する。応用例の一つとしてデータのシリアライズがある。

## GHC.Generics
パッケージ情報など付加的な情報も扱える

* [GHC.Generics - HaskellWiki](https://wiki.haskell.org/GHC.Generics)
